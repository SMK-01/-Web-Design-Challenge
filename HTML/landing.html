Summary: Latitude vs. X


The purpose of this project was to analyze how weather changes as you get closer to the equator.To accomplish this analysis,we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.After assembing the dataset, we used Matplotib to plot various aspects of the weather vs.latitude.Factors we looked at include: Temperature, Cloudiness, WindSpeed, and Humidity.This site provides the source data and Visualizations created as part of the analysis, as well as explanations and description of any trendss and correlations witnessed.
